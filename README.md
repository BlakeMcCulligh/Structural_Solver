# Structural Solver
A structural analysis program capable of 3D frame linear analyses, 3D frame cross-section optimization, and 2D truss cross-section and topology optimization. This program has been developed to assist the University of Waterloo Steel Bridge Design Team in designing lightweight steel bridges.

This program is <ins>not to be trusted</ins> with engineering calculations, and all results shall be verified by the user.  

This program is still in development, undergoing constant changes, and has many incomplete features. Please report any errors that occur while using the program.

## How To Run
To use the program, run main.__main__.py, and a window will open allowing inputs from Excel files and/or directly into the window, and different analyses to be run on your inputs. The 3D frame structure allows you to save your current inputs along with the results from a linear analysis, if run.

More details on how to use the program will be added as features are completed. If you have any questions, don't be afraid to ask at bmcculli@uwaterloo.ca
