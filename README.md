# Structural Solver
A structural analysis program capable of 3D frame linear analyses, 3D frame cross-section optimization, and 2D truss cross-section and topology optimization. This program has been developed to assist the University of Waterloo Steel Bridge Design Team in designing lightweight steel bridges.

This program is <ins>not to be trusted</ins> with engineering calculations, and all results shall be verified by the user.  

This program is still in development, with ongoing changes.
